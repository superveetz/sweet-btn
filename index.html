Sweet Button Demo

A jQuery plugin that gives you options for how your buttons should look and feel.

How It Works

  • Step 1. Include the /release/sweet-btn.js JavaScript file in your project

  • Step 2. Add sweet-btn and sweet-btn-{size} class names to any element:

    • - {size} options:
    • xl
    • lg
    • md
    • sm
    • xs

    Ex. <i class='sweet-btn sweet-btn-xl'></i>

  • Step 3. Add sweet-btn-{attribute-name}={value} attributes to any element:

    - There are currently 104 different attributes that can be supported simultaneously so the {attribute-name} options documentation is divded into two categories:

    • Appearance
    • &
    • Events

  • Step 4. Enable the Sweet-Button.js plugin by adding a <script> tag at the bottom of your page:
    <script>
      $(function () {
        $('.sweet-btn').sweetButton();
      });
    </script>

Appearance

  • Creating Shapes:
    sweet-btn-shape-{#}={type}

    • - {#} options:
    • 1
    • 2
    • 3
    • 4
    • 5
    Note: The {#} indicates the z-index or the stacking order for multiple shapes.
    • - {type} options:
    • hexagon
    • square
    • circle

    Shape Size:
    sweet-btn-shape-{#}-size={size}

    • - {size} options:
    • xs
    • sm
    • md
    • xl

    Shape Color:
    sweet-btn-shape-{#}-background={color}

    • - {color} options:
    • any css color

    Examples:

    <i class='sweet-btn sweet-btn-xl'
      sweet-btn-shape-1='hexagon'
      sweet-btn-shape-1-background='blue'
      sweet-btn-shape-1-size='md'

      sweet-btn-shape-2='square'
      sweet-btn-shape-2-background='red'
      sweet-btn-shape-2-size='sm'

      sweet-btn-shape-3='circle'
      sweet-btn-shape-3-background='gold'
      sweet-btn-shape-3-size='xs'>
    </i>

  • Adding Icons:
    sweet-btn-glyph-classes={classes}

    • - {classes} options:
    • any font awesome class
    Note: The {classes} parameter should always contain the class fa-fw in order to obtain a fixed width icon that will center nicely.

    Icon Size:
    sweet-btn-glyph-size={size}

    • - {size} options:
    • xs
    • sm
    • md
    • xl

    Icon Color:
    sweet-btn-glyph-color={color}

    • - {color} options:
    • any css color

    Examples:

    <i class='sweet-btn sweet-btn-xl'
      sweet-btn-glyph-classes='fa fa-fw fa-facebook-square'
      sweet-btn-glyph-size='sm'
      sweet-btn-glyph-color='skyblue'

      sweet-btn-shape-1='hexagon'
      sweet-btn-shape-1-background='blue'
      sweet-btn-shape-1-size='md'

      sweet-btn-shape-2='square'
      sweet-btn-shape-2-background='red'
      sweet-btn-shape-2-size='sm'

      sweet-btn-shape-3='circle'
      sweet-btn-shape-3-background='gold'
      sweet-btn-shape-3-size='xs'>
    </i>

  • Adding Text:

    • - simply add any text or HTML inside the element containing the sweet-btn class

    Example:

    • <i class='sweet-btn sweet-btn-xl'
        sweet-btn-glyph-classes='fa fa-fw fa-facebook'
        sweet-btn-glyph-size='sm'
        sweet-btn-glyph-color='skyblue'

        sweet-btn-shape-1='hexagon'
        sweet-btn-shape-1-background='blue'

        sweet-btn-shape-3-background='gold'

Events

  • Shape Events:
    sweet-btn-shape-{#}-{event-type}={classes}

    • - {#} options:
    • 1
    • 2
    • 3
    • 4
    • 5
    • - {event-type} options:
    • click
    • mouse-enter
    • mouse-leave
    • - {classes} options:
    • any css class
    Note: It is highly recommended to use Daniel Eden's animate.css library for animation classes.

    Icon Events:
    sweet-btn-glyph-{event-type}={classes}

    Examples:

    <i class='sweet-btn sweet-btn-xl'
      sweet-btn-shape-1='hexagon'
      sweet-btn-shape-1-background='blue'
      sweet-btn-shape-1-size='md'
      sweet-btn-shape-1-mouse-enter='wobble'

      sweet-btn-shape-2='square'
      sweet-btn-shape-2-background='red'
      sweet-btn-shape-2-size='sm'
      sweet-btn-shape-2-mouse-enter='shake'

      sweet-btn-shape-3='circle'
      sweet-btn-shape-3-background='gold'
      sweet-btn-shape-3-size='xs'>
      sweet-btn-shape-3-mouse-enter='bounce'
    </i>

  • Animation Duration:
    sweet-btn-shape-{#}-{event-type}-animation-duration={integer}

    Note: The {integer} parameter is in number of milliseconds, ie. 1000 for 1 second.
    • - {event-type} options:
    • click
    • mouse-enter
    • mouse-leave

    Animation Delay:
    sweet-btn-shape-{#}-{event-type}-animation-delay={integer}

    Animation Iteration Count:
    sweet-btn-shape-{#}-{event-type}-animation-iteration-count={integer}

    Examples:

    <i class='sweet-btn sweet-btn-xl'
      sweet-btn-shape-1='hexagon'
      sweet-btn-shape-1-background='blue'
      sweet-btn-shape-1-size='md'
      sweet-btn-shape-1-mouse-enter='wobble'
      sweet-btn-shape-1-mouse-enter-animation-delay='0'
      sweet-btn-shape-1-mouse-enter-animation-duration='3000'

      sweet-btn-shape-2='square'
      sweet-btn-shape-2-background='red'
      sweet-btn-shape-2-size='sm'
      sweet-btn-shape-2-mouse-enter='shake'
      sweet-btn-shape-2-mouse-enter-animation-delay='1000'
      sweet-btn-shape-2-mouse-enter-animation-duration='2000'

      sweet-btn-shape-3='circle'
      sweet-btn-shape-3-background='gold'
      sweet-btn-shape-3-size='xs'>
      sweet-btn-shape-3-mouse-enter='bounce'
      sweet-btn-shape-3-mouse-enter-animation-delay='2000'
      sweet-btn-shape-3-mouse-enter-animation-duration='1000'
    </i>
